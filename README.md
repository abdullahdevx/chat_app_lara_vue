Hello, I've developed this Chat Application using Laravel, Vue.js & Tailwindcss <br>
*** Technologies Used: ***<br>
LARAVEL: back-end developed in LARAVEL 10<br>
Vue.js: front-end is built in Vue.js 3<br>
TailwindCSS<br>

*** Features: ***<br>
Built with LARAVEL & Vue.js.<br>
User can start a chat<br>
User can send messages in realtime <br>
Built with Pusher for Realtime communication using web sockets.<br>

*** HOW TO RUN THIS PROJECT ON LOCAL HOST ***<br>
LARAVEL:<br>
Clone this project in your local computer by using "git clone (repo url)" Before Running this project<br>
open terminal run "composer install" create a .env file copy code from .envexample and paste into .env file Install and configue XAMPP (run mysql & localhost server)<br>
Create a pusher account and copy paste pusher credientials in .env file<br>
run "php artisan serve"<br>
migrate the database using "php artisan migrate" make sure to change the database creds in .env file<br>

Vue.js:<br>
Open terminal in the directory and run "npm install".<br>
Then npm run serve!<br>
