Hello, I've developed this Chat Application using Laravel, Vue.js & Tailwindcss
*** Technologies Used: ***
LARAVEL: back-end developed in LARAVEL 10
Vue.js: front-end is built in Vue.js 3
TailwindCSS

*** Features: ***
Built with LARAVEL & Vue.js.
User can start a chat
User can send messages in realtime 
Built with Pusher for Realtime communication using web sockets.

*** HOW TO RUN THIS PROJECT ON LOCAL HOST ***
LARAVEL:
Clone this project in your local computer by using "git clone (repo url)" Before Running this project
open terminal run "composer install" create a .env file copy code from .envexample and paste into .env file Install and configue XAMPP (run mysql & localhost server)
Create a pusher account and copy paste pusher credientials in .env file
run "php artisan serve"
migrate the database using "php artisan migrate" make sure to change the database creds in .env file

Vue.js:
Open terminal in the directory and run "npm install".
Then npm run serve!
